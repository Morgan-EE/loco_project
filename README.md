# 5. 실행 방법 (Git Bash 기준)
## 프론트엔드
### frontend 경로 이동
```bash
cd frontend
```
### npm 설치
```bash
npm install
```
### npm 실행
```bash
npm run dev
```
---
## 백엔드
### backend 디렉토리에서 빌드 진행
```bash
cd backend
./gradlew clean build
```
or
```bash
./gradlew build
```
### 실행 파일 (jar)이 있는 디렉토리로 이동 및 jar 파일 실행
```bash
cd backend/build/libs
java -jar loco_project-0.0.1-SNAPSHOT.jar
```
